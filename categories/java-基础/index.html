变量和方法

静态变量，被队友对象共享，内存中只有一个副本，仅仅在类加载的时候被初始化。
是属于这个类的，不属于某个变量，被类中所有变量共享.

如果方法执行的操作不依赖于其类的各个变量和方法，请将其设置为静态（这将使程序的占用空间更小）。 否则，它应该是非静态的。
静态方法在类加载的时候就存在了，它不依赖于任何实例，所以静态方法必须有实现，也就是说它不能是抽象方法（abstract）。
