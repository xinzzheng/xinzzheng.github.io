ADE Commands
2019-04-03|Oracle Tools

常用的ADE Commands:

* cd $ADE_VIEW_NAME
* cd $ADE_VIEW_ROOT
* ade okinit
* setenv ADE_SITE ade_fusion_slc
* ade createview -series <Series Name> (-label <label name> or -latest) <view name/workspace name on local>    -------> Create workspace and checkout
* ade createview scm_orchinfra_view -series FUSIONAPPS_PT.INTSCMC_LINUX.X64 -latest 
* ade useview <view name>   ------->Start using the view/workspace
*     jdev &  ------->open Jdeveloper with the view created in step 2. we need to import the code manually.
* ade begintrans <bug number> -no_restore ------->/ create trans file on the view. It will take time so we will start jdev.
* ade ci <filename>
* ade ci -all or ade ciall ------->to checkin all files
* 

* ade begintrans ranathi_scmc_trnx -------> Begins transaction
* ade savetrans <trans_name> ------->/ after this others can view the trans file.
* ade describetrans <trans_name> -------> it will tell how many files we changed .
* ade renametrans -bug <bug number>   ------> change bug number
* ade destroytrans -force <trans_name> ------->local changes will be destroyed and trans file will be deleted.
* ade tkreview <trans_name> ------->To see a particular transaction for changes
* ade tkreview -view  ------->To see a particular transaction along with the file that are not checked-in.
* ade tkvtree <file>
* ade lsco
* ade unco <filename>  / ade unco -all ------->the changes will be unbranched  from trans file, but changes will be still there on our local. To branch again change the file again and it will be associated to transaction.
* ade fetchtrans(get saved trans) /grabtrans(get ci transactions) <trans_name> -------> to merge other person trans file to our local.
* ade grabtrans <trans_name>
* ade sharetrans <trans_name>
ade unbranch <fullpath>    ------> rollback the checkin, or cancel the checked in files from txn
* ade showseries | grep SCMC  -------> Shows all serires

* ade lsviews -------> To view all our views
*     vncconfig &  -------> To copy from windows to VM
* ade lshistory <filename> |grep Merges  ------> To get History of file
* ade showlabels -series FUSIONAPPS_11.1.1.5.1_LINUX.X64 ------>  Shows all labels in a series
* ade exec /usr/local/nde/ade/util/adhoc_query_scripts/list_merges_to_branch.pl -p FUSIONAPPS -b st_fusionapps_11.1.1.5.1 -d 30 -sort_by  // Get transactions of past 30 days
* ade mkelem <file-name> ----> To add file to branch
* ade rm <file-name>
* ade pwv          ------> Display info for the named view; defaults to the current view
* ade pwd          ------> displays view-relative path, otherwise displays absolute path.
* ade destroyview <view name>
