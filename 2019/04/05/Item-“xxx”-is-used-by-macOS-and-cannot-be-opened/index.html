2019-04-05|Mac系列

问题描述

Item “xxx” is used by macOS and cannot be opened.

问题原因

macOS 默认挂载 ntfs 是只读的，当然，我们有办法使用命令行重新挂载它来变成可读写。不过这容易造成 ntfs 权限错误，结果就是插到另一台电脑上的时候，这个文件可能变成灰色。

使用 xcode-select –install 来安装命令行工具，安装完成后，我们可以使用下面的命令来查看这个文件的属性：

解决办法

SetFile -c "" -t "" a.zip
